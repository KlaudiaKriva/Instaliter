(Bearer Token),(JWT - JSON Web Token)
Token needs to be set in the header of every request except in registration and login (just to be sure :D).
If you request an image |example: http://localhost:5005/images/kmxalsmfekasdopmp5.jpg| token is not used.
                                  http://localhost:5005/thumbnails/kmxalsmfekasdopmp5.jpg
One token is valid for 30 days, after 30 days the user have to log in again.
You have to store id of the user and the token in a local storage or somewhere else.

Code example for a request with token:
